# Ontology Content

This directory is being re-based around the traditional astrology research report that now serves as The Ark's doctrinal source of truth.

## Target layering
- traditional core: planets, signs, houses, sect, rulers, dignities, joys, lots, and timing doctrine
- later traditional extensions: medieval timing and reception systems added with clear labels
- optional modern overlays: Jungian mappings, imaginal prompts, and other reflective additions
- app synthesis: explanatory bridges that help the UI teach without pretending modern overlays are ancient doctrine
